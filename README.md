

# 2021-Connect

## 서비스 소개
* 서비스명 : 커넥트
* 주제 : 독거노인과 사회를 이어주는 어플

## 팀 소개
* 팀명 : Traveller(여행자)
* 팀장 : 김보경
* 팀원 : 윤세림, 박지현

## 팀 활동 규칙
* 팀회의시간 : 일요일 오후 12시  
* 블로그 작성 및 발표 순서 : 김보경 → 박지현 → 윤세림 

## 사용할 기술 스택
* Flutter
* node.js
* MySQL
* Firebase

## 대략적인 목표
*플러터의 기본 문법 공부부터 스터디를 시작하여 프론트엔드(앱)을 구성하고, DSC Duksung에서 사용해보았던 node.js를 실제로 적용해본다. 
이 프로젝트는 노인분들에게 어플의 접근성이 쉽고 user-friendly app을 만드는 것이 목표이다.
